How to use it:

  • Load images or video (drag'n'drop them or click the "+" slot).
  • Select a loaded image or video by clicking on it.
  • Import existing effects using the left menu.
  • If any, set parameters using the left menu.
  • Enjoy!

Known bugs:

  • Firefox throws an error when double-clicking the output, but the format/type pair is correct.
  • When loading mobile version then turning to desktop one, bug when resizing the menu.
